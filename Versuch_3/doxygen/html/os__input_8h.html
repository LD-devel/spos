◆ os_getInput()

uint8_t os_getInput ( void  )

A simple "Getter"-Function for the Buttons on the evaluation board.

Returns
The state of the button(s) in the lower bits of the return value.
example: 1 Button: -pushed: 00000001 -released: 00000000 4 Buttons: 1,3,4 -pushed: 000001101
